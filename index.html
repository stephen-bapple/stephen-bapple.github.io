Current Project

Jokes on the Go

This project was mainly started in order to pratice my UI skills, Golang and really learn Kotlin. It's mostly an excuse, but I love to tell corny dad-type jokes and struggle to keep track of which ones I've told to whom. So, I'll build an app for that.

Hightlights / What's working so far

  • Uses gRPC to communicate between client and server
  • gRPC stubs published automatically on tag in Github Actions, downloaded in separate Android repo
  • Jetpack Compose provides the UI, although it doesn't do much yet
  • Server and client use tls and client is authenticated with an API key

Planned features

  • Host server on real publically accessible domain name
  • Use a real certificate authority to avoid self-signed cert scripting
  • Use postgres to store a lot of jokes
  • Implement tagging and audience tracking
  • Add "Add API key" feature to Android app
  • Publish android APK for easy install
  • Build a more functional and visually pleasing UI
  • ...Potentially even more features once the app is fleshed out
