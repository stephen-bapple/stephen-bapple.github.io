Current Project

Jokes on the Go

This project was mainly started in order to practice both my UI skills and Golang as well as really learn Kotlin. It's mostly an excuse, but I love to tell corny dad-type jokes and struggle to keep track of which ones I've told to whom. So, I'll build an app for that.

Right now the app and server are both very barebones. I chose to get the connection working first so in order to prove out all relevant technologies. In the past I've encountered difficulties getting gRPC or imitators to work with certain tech stacks, and changing API technology can be painful, so I wanted to remove that risk. I particularly love this pattern for development as stubbing out server responses is a great way to parallelize development work.

Highlights / What's working so far

  • Uses gRPC to communicate between client and server
  • gRPC stubs are published automatically on tag in Github Actions, to be downloaded in separate Android repo
  • Jetpack Compose provides the UI, although it doesn't do much yet
  • Server is hosted on a Raspberry PI, with the gRPC API exposed at jokesonethego.com
  • Server uses certbot/letsencrypt for TLS, and the client is authenticated with an API key
  • Server is continually built and deployed via Github Actions, a webhook, and systemd wizardry
